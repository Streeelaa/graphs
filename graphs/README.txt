

///////////////////////////////////////////////////////////////////////////////////////////////////
Для запуска просто собрать проект и запустить его))))))

Реализовано три графика:
1) Парабола. Отображается сразу после запуска, переключается при помощи кнопки "F1" на клавиатуре
2) Линейная функция. Отображается после нажатия кнопки "F2" на клавиатуре
3) функция квадратного корня. Отображается после нажатия кнопки "F3" на клавиатуре

Если нужно поменять коэфициенты у функций, то нужно перейти во graphsView -> OnDraw и в реализации swith case поменять коэфициенты на 
те, которые требуются пользователю (если у функции есть какие либо ограничения при построении, то это стоит учитывать, 
задавая коэфициенты, иначе могут возникнуть неточности при построении графика)

///////////////////////////////////////////////////////////////////////////////////////////////////

В MainFrm.cpp создается окно размером 1200 на 1080 пикселей

В graphsView в Ondraw, реализовано построение двух координатных осей с их шагом(1 шаг = 20 пикселей), далее идет построение
самих графиков. Реализация и добавление новых графиков осуществляется через func.h с использованием ДИНАМИЧЕСКОГО ПОЛИМОРФИЗМА!!!)))

в graphsView.h находятся обработчики событий, которые связывают нажатие клавиш с построением графиком, то есть,
при нажатии определенной клавиши обновляется окно и перересовывается график.
